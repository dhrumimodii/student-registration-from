# Flutter Student Registration Form

A simple Flutter application that demonstrates form validation using `Form` and `TextFormField`.

##  Features
- Student registration form UI  
- Input validation for name, email, phone, and password  
- Error messages for invalid inputs  
- Success message on valid submission  

##  Tech Stack
- Flutter  
- Dart  

##  Form Validation Rules
- Name: Required  
- Email: Must be a valid email format  
- Phone: Must be a 10-digit number  
- Password: Minimum 6 characters  

##  Purpose
This project is built to practice Flutter form handling, validation, and user input management.
